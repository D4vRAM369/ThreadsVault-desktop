## Resumen de Cambios por Archivo

### NUEVOS ARCHIVOS

#### `src/components/CategoryManager.svelte` (+370 líneas)
Componente reutilizable extraído de `CategoryScreen.svelte`. Incluye:
- CRUD: crear, editar inline, eliminar con confirmación
- Emoji picker (grid de emojis frecuentes)
- Color swatches (paleta de colores predefinidos)
- Drag & Drop con HTML5 DnD API + visual feedback
- Prop `compact={boolean}` para modo embebido en Settings

#### `src/lib/utils/post-extractor.ts` (+157 líneas)
Utilidad que obtiene metadatos de una URL de Threads:
- oEmbed API (Threads/Instagram)
- Open Graph meta tags
- Twitter meta tags
- Extrae: title, text, images, videos, media items[]

### ARCHIVOS MODIFICADOS

#### `src/components/PostCard.svelte`
- Agregado `role="button"` + `aria-label` al `<article>`
- Badge de media: muestra `🖼️ N` o `🎬 N` cuando `post.media` existe
- Eliminado `stopPropagation` del row de confirmación de borrado
- Eliminado import no usado `cleanThreadsUrl`

#### `src/lib/storage/dexie.ts`
- `SCHEMA_VERSION` 1 → 2
- Categorías por defecto ahora incluyen `emoji` y `order`
- `getCategories()` ahora ordena por campo `order`

#### `src/lib/storage/sqlite.ts`
- `SCHEMA_VERSION` 1 → 2
- Nuevas columnas en `posts`: `previewVideo`, `extractedText`, `canonicalUrl`, `mediaJson`
- Nuevas columnas en `categories`: `emoji`, `sortOrder`
- `_runMigrations()` + `_ensureColumn()` para migraciones seguras
- `_rowToPost()` helper que deserializa `mediaJson` → `PostMedia[]`
- `getCategories()` ordena por `sortOrder`
- `savePost()` y `saveCategory()` manejan todos los nuevos campos

#### `src/lib/stores/vault.ts`
- Motor de búsqueda con scoring (`scorePostAgainstTerms`, `tokenizeQuery`, `normalizeText`)
- `allHashtags` ahora extrae de `extractedText` y `previewTitle` además de `note`
- `hashtagStats`: derived store con `{tag, count}[]` ordenado por frecuencia
- `filteredPosts`: hashtag filter busca en todos los campos de texto
- `filteredPosts`: búsqueda multi-término con ranking por relevancia

#### `src/lib/types.ts`
- Añadido: `PostMediaType`, `PostMedia` interfaces
- Post extendido: `previewVideo?`, `extractedText?`, `canonicalUrl?`, `media?: PostMedia[]`

#### `src/routes/CategoryScreen.svelte`
- Reducida de ~280 líneas inline → thin wrapper que importa `<CategoryManager />`

#### `src/routes/PostDetailScreen.svelte`
- Import `PostMedia`
- `downloadMedia()` helper
- Muestra `previewTitle`, bloque `extractedText`, galería de media con botones de descarga

#### `src/routes/SettingsScreen.svelte`
- `<CategoryManager compact={true} />` embebido directamente
- Mejoras responsive con breakpoints `sm:`

#### `src/routes/ShareScreen.svelte`
- Integra `extractPostData()` al guardar: fetch de metadatos antes de persistir
- Todos los nuevos campos del Post se populan al guardar
- Botón cambia a "Analizando y guardando..." durante extracción

#### `src/routes/VaultScreen.svelte`

## Stack

- **Framework:** Svelte 5 (runes: `$state`, `$derived`, `$effect`, `$props`)
- **Build:** Vite + TypeScript + Tailwind v4
- **Storage PWA:** Dexie.js (IndexedDB)
- **Storage Desktop:** tauri-plugin-sql (SQLite) — Tauri v2
- **Fonts:** Raleway ExtraBold (brand), Syne 600-800 (display), DM Sans (body)
- **Design:** Glassmorphism, blob animations, CSS variables, dark theme
- **Colores:** Primary `#7C4DFF` (purple), Secondary `#00BCD4` (cyan), Tertiary `#26A69A` (teal)
